Client Configuration

Configure your AI coding tools to use this gateway. Each tool below connects via the OpenAI-compatible endpoint http://localhost:10086/v1.

◆ OpenCode ▶

Add to your OpenCode configuration file (e.g. opencode.json):

{
  "provider": {
    "fmg": {
      "name": "Free Model Gateway",
      "options": {
        "baseURL": "http://localhost:10086/v1",
        "apiKey": "fmg-local"
      },
      "models": {
        "auto": { "name": "Auto (Smart Routing)" }
      }
    }
  }
}

Then select fmg/auto as your provider in the model picker.

◆ Claude Code ▶

Set environment variables before launching Claude Code:

export ANTHROPIC_BASE_URL="http://localhost:10086"
export ANTHROPIC_API_KEY="fmg-local"

Or use OpenAI-compatible mode:

export OPENAI_BASE_URL="http://localhost:10086/v1"
export OPENAI_API_KEY="fmg-local"
◆ Trae ▶

In Trae settings:

  1. Go to Settings → Models → Custom Provider
  2. Set API Endpoint: http://localhost:10086/v1
  3. Set API Key: fmg-local
  4. Select model auto for smart routing
◆ Qoder ▶

In Qoder settings:

  1. Open Settings → AI Provider
  2. Choose OpenAI Compatible
  3. Set Base URL: http://localhost:10086/v1
  4. Set API Key: fmg-local
  5. Set Model: auto
◆ Hermes ▶

In Hermes configuration:

  1. Open Settings → Model Providers
  2. Add a new OpenAI Compatible provider
  3. Set Endpoint: http://localhost:10086/v1
  4. Set API Key: fmg-local
  5. Select model auto
